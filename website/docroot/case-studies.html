1&1 Internet AG Websites

1&1 is an international web hosting company and internet access provider.

1&1 uses Pustefix to power nearly all of their websites, where customers order the 1&1 products. They are available in five languages (German, British English, American English, French and Spanish) and provide ordering processes for all 1&1 products. The different websites are visited by more than 20.000.000 users per month; the applications contain of up to 500 different pages. All of these pages are rendered individually for each visitor; no static HTML pages are served.

1&1 focusses on a very modular application structur, making heavy use of the Pustefix module system. The different applications share more than ten modules, ranging from logging facilities, to business logic or complete AJAX components. This allows the developers to implement new core features for the applications and quickly deploy them to the different projects where they are needed. 1&1 does not only use Java components in modules, JavaScript and CSS libraries are also deployed using the Pustefix module system integrated in a custom Maven build.
